# FrogSide RP FastDL

This repository is a GitHub-ready FastDL mirror for FrogSide RP client content.

The repository root is intended to be served directly as the FastDL web root. Asset paths therefore stay in their Garry's Mod layout, for example:

- `materials/...`
- `models/...`
- `sound/...`
- `particles/...`

Generated source:

- `outputs/frogside_fastdl_github_ready_20260619_161000`

Notes:

- Files are `.bz2` compressed for FastDL.
- The oversized map archive was excluded from this GitHub-safe upload because it is above GitHub's normal single-file limit.
- Keep gameplay Lua, paid addon source, configs, databases, credentials, and server-only files out of this repository.
